# GLP-1 Guide 配色方案提案

## 研究总结

根据医疗健康行业网站配色研究：
- **85%的医疗公司使用蓝色** - 传达信任、可靠、冷静
- **22%使用绿色** - 象征治愈、成长、和谐
- **需要避免过于冷色调** - 可加入暖色中性色（beige, coral）
- **必须确保可访问性** - WCAG对比度标准

## 当前问题分析

1. **颜色混用不统一**：medical-blue、emerald、teal、red、amber等混用，缺乏规则
2. **主次色不明确**：没有清晰的primary/secondary/accent层次
3. **渐变不协调**：多个渐变同时使用，视觉混乱
4. **语义化不足**：相同功能使用不同颜色

## 方案一：经典医疗蓝（Classic Medical Blue）

### 设计理念
遵循医疗行业标准，以蓝色为主，绿色为辅，传达专业、可信赖的形象。

### 配色方案

```typescript
Primary (主色 - 60%):
- Blue 600: #2563EB (主要CTA、链接、重要元素)
- Blue 500: #3B82F6 (hover状态)
- Blue 700: #1D4ED8 (active状态)

Secondary (辅助色 - 30%):
- Emerald 600: #10B981 (成功、节省、正面信息)
- Emerald 500: #34D399 (hover)
- Emerald 700: #059669 (强调)

Accent (强调色 - 10%):
- Teal 500: #14B8A6 (特殊功能、数据可视化)
- Amber 500: #F59E0B (警告、注意)
- Red 500: #EF4444 (错误、危险)

Neutral (中性色):
- Gray 50-900: 标准灰度，用于文本、背景、边框
```

### 使用规则
- **Primary Blue**: 主要按钮、导航、品牌元素、重要链接
- **Secondary Emerald**: 价格节省、成功消息、正面指标、健康相关
- **Accent Teal**: 数据图表、特殊功能按钮
- **Accent Amber**: 警告提示、注意事项
- **Accent Red**: 错误消息、危险操作

### 适用场景
- 专业医疗信息平台
- 需要建立强烈信任感的场景
- 面向中老年用户群体

## 方案二：现代医疗蓝绿（Modern Blue-Green）

### 设计理念
平衡蓝色和绿色，营造现代、健康、积极的氛围，适合健康科技平台。

### 配色方案

```typescript
Primary (主色 - 50%):
- Blue 600: #2563EB (主要CTA、导航)
- Blue 500: #3B82F6 (hover)

Secondary (辅助色 - 40%):
- Emerald 600: #10B981 (健康、节省、正面信息)
- Emerald 500: #34D399 (hover)
- Teal 500: #14B8A6 (数据、图表)

Accent (强调色 - 10%):
- Amber 500: #F59E0B (警告)
- Red 500: #EF4444 (错误)
- Purple 500: #A855F7 (特殊功能)

Neutral:
- Slate 50-900: 更现代的灰色调
```

### 使用规则
- **Primary Blue**: 主要操作、导航、品牌
- **Secondary Emerald**: 健康相关内容、价格节省、成功状态
- **Secondary Teal**: 数据可视化、科技感元素
- **Accent**: 仅用于特殊状态（警告、错误）

### 适用场景
- 健康科技平台
- 面向年轻用户群体
- 需要传达创新、现代感

## 方案三：温暖医疗（Warm Medical）

### 设计理念
蓝色+暖色中性色，避免冷色调，营造温暖、友好、可接近的氛围。

### 配色方案

```typescript
Primary (主色 - 55%):
- Blue 600: #2563EB (主要CTA、导航)
- Blue 500: #3B82F6 (hover)

Secondary (辅助色 - 30%):
- Emerald 600: #10B981 (健康、节省)
- Sage Green 500: #87C5A4 (温和的绿色，用于背景)

Accent (强调色 - 15%):
- Coral 400: #FB7185 (温暖的点缀，用于特殊CTA)
- Amber 500: #F59E0B (警告)
- Red 500: #EF4444 (错误)

Warm Neutral:
- Beige 50: #FEFCFB (背景)
- Warm Gray 100-900: 带暖调的灰色
```

### 使用规则
- **Primary Blue**: 主要操作、导航
- **Secondary Emerald**: 健康、节省
- **Sage Green**: 背景渐变、温和提示
- **Coral**: 特殊CTA、吸引注意
- **Warm Neutral**: 文本、卡片背景

### 适用场景
- 需要降低医疗焦虑感的平台
- 面向初次使用医疗服务的用户
- 心理健康、健康咨询类内容

```typescript

- Blue 400: #60A5FA (浅蓝，背景、卡片)

Secondary (辅助色 - 30%):
- Emerald 600: #10B981 (成功、节省)
- Emerald 400: #34D399 (浅绿，背景)

Accent (强调色 - 10%):
- Amber 500: #F59E0B (警告)
- Red 500: #EF4444 (错误)
- Indigo 500: #6366F1 (特殊功能)

Neutral:
- Gray 50-900: 标准灰度
```

### 使用规则
- **Navy**: 品牌标识、导航栏、重要标题
- **Blue 600**: 主要CTA按钮
- **Blue 400**: 背景渐变、卡片背景
- **Emerald**: 价格节省、成功消息
- **Accent**: 仅用于状态提示

### 适用场景
- 专业医疗信息平台
- 需要传达权威性的内容
- 面向医疗专业人士

### 使用规则

- 健康信息平台（最推荐）

## 实施建议

### 统一规则（所有方案通用）

1. **60-30-10规则**
   - Primary: 60% 的使用
   - Secondary: 30% 的使用
   - Accent: 10% 的使用

2. **语义化颜色**
   - 成功/节省 → Secondary色
   - 警告 → Amber
   - 错误/危险 → Red
   - 信息/数据 → Teal或Primary

3. **渐变使用**
   - 每个页面最多1-2个渐变
   - 渐变方向统一（通常从上到下或从左到右）
   - 使用同色系渐变，避免跨色系

4. **按钮层次**
   - Primary按钮：Primary色，实心
   - Secondary按钮：Primary色，边框
   - Tertiary按钮：中性色，文本链接样式

5. **卡片设计**
   - 背景：白色或浅色（Primary/Secondary 50-100）
   - 边框：Primary/Secondary 200-300
   - Hover：Primary/Secondary 100背景
