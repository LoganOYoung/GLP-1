## 3. 图片 SEO 规范（已落实）

- **Alt**：每张图都有描述性 alt，含关键词（GLP-1、calculator、savings、legitimacy 等）且语句通顺。
- **尺寸**：代码中统一使用 `width` / `height`，与上表一致，减少 CLS。
- **优先级**：首屏 Hero 与 LCP 图使用 `priority`，其余默认 lazy。
- **格式**：统一使用 .webp，路径与文件名语义化（banners / inline / drugs / infographics / logos）。
- **占位**：ImagePlaceholder 在 404 或加载失败时显示渐变+文案，避免裂图。

---

## 4. 生成/获取建议（供设计或 AI 出图）

- **Banners**：横版 16:9 或 2.35:1，健康/医疗风格，少文字，可带 GLP-1、计算器、保险、FAQ、合法、替代、TrumpRx、实验室等主题。
- **Inline**：生活化、信任感（人物、药房、储蓄、支持、路径、申请、透明、生活方式），色调与站内一致。
- **Drugs**：药品包装或官方风格图，合规无夸大。
- **Logos**：站标，可 SVG 或 32×32 / 64×64 导出 webp。
- **Infographics**：品牌/复方/口服三条路径的简洁图示或卡片图。

将生成好的文件按上表路径放入 `public/images/` 对应子目录即可；缺失时由 ImagePlaceholder 显示占位。
