# 视觉元素增强计划

## 当前问题
网站目前只有文字和图标，缺少：
- **图片/Banner**: Hero区域、页面顶部缺少视觉冲击力
- **信息图表**: 数据可视化、流程图表、对比图表
- **表格优化**: 现有表格需要更好的视觉设计
- **插图**: 医疗相关插图、流程图、示意图

## 需要添加视觉元素的位置

### 1. 首页 (app/page.tsx)

#### Hero Banner
- **位置**: Hero区域右侧（目前只有图标网格）
- **内容**: 
  - 医疗健康相关的插图/图片
  - 或者数据可视化图表（成本对比、节省金额等）
- **尺寸**: 大尺寸，吸引注意力

#### 信息图表
- **"We help with" 区域**: 每个卡片添加小图标或插图
- **工具区域**: 添加使用流程图表
- **内容中心区域**: 添加对比图表

### 2. Alternatives 页面

#### Banner
- **位置**: 页面顶部Hero区域
- **内容**: GLP-1药物对比示意图、或选择流程图

#### 信息图表
- **Path Comparison Table**: 增强表格设计，添加颜色编码、图标
- **价格对比图表**: 柱状图/折线图显示价格对比
- **副作用对比**: 可视化副作用对比图表
- **剂量转换工具**: 添加视觉化的注射器示意图（已有SyringeVisualization，需要增强）

### 3. Calculator 页面

#### Banner
- **位置**: 计算器顶部
- **内容**: 计算流程示意图

#### 信息图表
- **结果展示**: 
  - 成本对比柱状图
  - PA成功率饼图
  - 年度节省金额可视化
- **步骤指示**: 添加进度可视化

### 4. Cost & Insurance 页面

#### Banner
- **位置**: Hero区域
- **内容**: 节省金额示意图、保险流程图表

#### 信息图表
- **折扣卡对比**: 表格+图表结合
- **保险申诉流程**: 流程图
- **TrumpRx状态**: 地图可视化（已有，需要增强）
- **HSA/FSA资格**: 资格流程图

### 5. 其他页面

#### Legitimacy页面
- **合规流程图**: 503A vs 503B对比图（已有ComplianceFlowDiagram，需要增强）
- **警告标志**: 视觉化的警告图标

#### TrumpRx页面
- **地图可视化**: 州状态地图（已有TrumpRxStateMap，需要增强）
- **政策变化时间线**: 时间线图表

## 实施策略

### 方案A: 使用占位图片服务（快速）
- 使用 Unsplash, Pexels 等免费图片
- 使用 placeholder.com 生成占位图
- **优点**: 快速实施
- **缺点**: 可能不够专业

### 方案B: 生成医疗相关图片（推荐）
- 使用 AI 生成工具创建医疗健康相关插图
- 创建信息图表和数据可视化
- **优点**: 专业、定制化
- **缺点**: 需要时间生成

### 方案C: SVG图表库（最佳）
- 使用 Recharts, Chart.js, D3.js 创建数据可视化
- 使用 SVG 创建流程图和信息图表
- **优点**: 专业、可交互、SEO友好
- **缺点**: 需要开发时间

## 技术实现

### 图片存储
- 创建 `/public/images/` 目录
- 子目录结构：
  - `/public/images/banners/` - Banner图片
  - `/public/images/infographics/` - 信息图表
  - `/public/images/charts/` - 数据图表
  - `/public/images/illustrations/` - 插图

### 图表库选择
- **Recharts**: React图表库，适合数据可视化
- **Chart.js + react-chartjs-2**: 功能强大
- **SVG**: 自定义流程图和信息图表

### Next.js Image优化
- 使用 `next/image` 组件
- 添加图片优化和懒加载
- 支持WebP格式
